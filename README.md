🛒 E-Commerce Sales Analytics Project

📋 Project Overview
This project is an end-to-end data analysis solution designed to extract, clean, and visualize E-commerce sales data. The goal is to provide actionable insights into revenue trends, customer behavior, and regional performance to help stakeholders make data-driven decisions.

🛠️ Tech Stack
Data Processing: Python (Pandas)

Database Management: MySQL

Data Visualization: Power BI

Language: SQL, Python (Jupyter Notebook)ts.
